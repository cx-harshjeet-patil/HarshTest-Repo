# Checkmarx Azure DevOps Testing

This repository is used to test:

- Azure DevOps YAML Pipelines
- Checkmarx One AST Plugin
- Pull Request Decoration 

## How to Use

1. Connect it to Azure DevOps
2. Install the Checkmarx AST ADO plugin
3. Create a Checkmarx service connection
4. Update the pipeline with your agent pool and service connection
5. Create a PR to `main` and observe PR decoration
